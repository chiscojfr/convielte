convieLte

A programing language to easily convert between units and more...

  • Github

Don't spend time doing conversions using traditional formulas.

Get them fast with ConvieLte!

Our motivation...

Have a system which we can convert any kind of units, and with that solve physics and electronic circuits problems. All of this in the same platform.

Program features:

  • Convert between units of any type of measurement
  • Store these values in variables to use in other funtions
  • Resolve basics problems of physics and electrical engineering.
